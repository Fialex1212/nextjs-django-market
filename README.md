# **Clothing Marketplace**

- **Goal:** Create a marketplace for clothing that offers a variety of styles and options.
- **Features:**
  - **User Registration:** Users can sign up to receive promotions (e.g., 20% off first order).
  - **Product Categories:** 
    - Men’s Clothing: T-shirts, Shorts, Shirts, Hoodies, Jeans
    - Women’s Clothing: T-shirts, Dresses, Casual, Formal
    - Accessories: Various styles and options
  - **Shopping Cart:** Users can add items and view order summaries, including prices, discounts, and delivery fees.
  - **Product Filtering:** Allow users to filter products by size, color, style, and rating.
  - **Reviews:** Users can leave feedback and rate products.
  - **Promotions:** Include promo codes for discounts on orders.
  
- **Design Elements:**
  - Emphasis on high-quality images and product descriptions.
  - User-friendly interface with easy navigation.

---



## **Tech Stack:**
  - Frontend: React, Next.js, Tailwind CSS
  - Backend: Django (REST Framework), FastAPI
  - Database: SQL (e.g., PostgreSQL or MySQL)

---
## **Installation**
Clone my project
```cmd
    git clone https://github.com/Fialex1212/nextjs-django-market.git
```

### **Frontend**
Run the frontend

```bash
  cd frontend
  cd market
  yarn install
  yarn dev
```

### **Backend**
Run the backend

```cmd
  cd backend
  cb base
  python -m venv venv
  .\venv\scripts\activate
  pip install requirements.txt
  python manage.py makemigrations
  python manage.py migrate
  python manage.py runserver
```
